# AuroraMusic3
Versao 3.0 do Music Player com novos recursos. 
**Descrição**
Plataforma de streaming e gerenciamento de áudio baseada em arquitetura PHP MVC, projetada para alta performance em hardware legado e sistemas Linux. O projeto prioriza baixo consumo de recursos e respostas rápidas de interface.

**Funcionalidades Atuais**
* **Controles de Reprodução:** Funções padrão de Play, Pause, Avançar e Retroceder faixas.
* **Gerenciamento de Playlist:** Listagem automática de arquivos e organização de fila de reprodução.
* **Playlists Personalizadas:** Criação e gestão de listas de reprodução customizadas pelo usuário.
* **Suporte Bluetooth:** Detecção e suporte para dispositivos de áudio e fones de ouvido sem fio.
* **Interface Adaptável:** Layout otimizado para desktop e dispositivos móveis via CSS Grid.

**Funcionalidades em Desenvolvimento**
* **Modo Offline:** Sistema de download de faixas para reprodução sem conexão à internet.
* **Módulo de Pagamento:** Integração com gateway PagSeguro para serviços premium ou assinaturas.

**Requisitos Técnicos**
* Servidor Web (Apache/Nginx)
* PHP 8.x (Arquitetura MVC)
* MySQL/MariaDB
* Otimizado para execução em ambientes Fedora Linux.
