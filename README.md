# Aurora Music 3.0

## Descrição
Plataforma de streaming e gerenciamento de áudio baseada em arquitetura **PHP MVC**, projetada para alta performance em hardware legado e sistemas Linux.

O projeto prioriza:
- Baixo consumo de recursos
- Respostas rápidas de interface

---

## Funcionalidades Atuais

### Reprodução
- Play
- Pause
- Avançar faixa
- Retroceder faixa

### Gerenciamento de Playlist
- Listagem automática de arquivos
- Organização da fila de reprodução

### Playlists Personalizadas
- Criação de playlists
- Gestão de listas customizadas pelo usuário

### Suporte Bluetooth
- Detecção de dispositivos de áudio
- Compatibilidade com fones sem fio

### Interface Adaptável
- Layout responsivo
- Otimizado para desktop e mobile
- Uso de CSS Grid

---

## Funcionalidades em Desenvolvimento

- Modo Offline  
  Sistema de download de faixas para reprodução sem conexão à internet

- Módulo de Pagamento  
  Integração com gateway PagSeguro para:
  - Serviços premium
  - Assinaturas

---

## Requisitos Técnicos

- Servidor Web (Apache ou Nginx)
- PHP 8.x (Arquitetura MVC)
- MySQL ou MariaDB
- Otimizado para execução em ambientes Fedora Linux
